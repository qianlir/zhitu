# 上海市民办永昌中学 (民办永昌中学)

> 上海市民办永昌中学创办于2015年，位于金山区山富西路99号，隶属于上海永昌教育集团，与创办于1995年的黄浦区民办永昌学校同为集团成员校。金山校区占地60亩，建有1万平方米教学大楼及高尔夫练习场、田径运动场、足球场、游泳池等设施，是一所全日制寄宿制完全中学。学校实行小班化教学，以高尔夫为特色提升综合素质，数学和外语为教学两大基石，VEX机器人、民族打击乐、书法、篮球为四大支柱特色项目。集团实行五项统一：统一理念、统一管理、统一课程、统一调配、统一评价。开设加拿大国际高中课程项目。
