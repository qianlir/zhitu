# 上海民办行中中学 (民办行中中学)

> 上海民办行中中学创办于2001年，是受宝山区教育局和行知教育集团双重领导的民办完全中学，依托上海市实验性示范性高中行知中学的优质教育资源。2023年7月复设，获批上海市民办普高办学资质。学校以'知行合一、守正创新、学贯中外'为办学理念，坚持'科技特色、外语特长'的办学特色。初三优秀毕业生在同等条件下可优先推荐自主录取升入行知中学。2024年复招后迁至新校址，紧邻地铁7号线祁华路站，学校占地约3万平方米。国际部开设A-level、VCE和AP三类国际课程，提供国内外双通道升学模式。2025年全市招生240名。
