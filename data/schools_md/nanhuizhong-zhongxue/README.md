# 上海市南汇第一中学 (南汇一中)

> 上海市南汇第一中学前身为南汇听潮中学，创建于1992年8月，2004年迁入原上海市南汇中学旧址并更名，2010年被命名为浦东新区实验性示范性高中。学校是一所完全中学，拥有卫星东路和听潮南路两个校区，初中部36个班、高中部18个班。学校依托校园内明代古城墙及清代孔庙大成殿，以'德能兼备、心蕴儒雅'为办学理念，将优秀传统文化培育现代合格公民作为办学特色，文化底蕴深厚。中考成绩在浦东新区名列前茅，高考录取率在区内普高中领先。
