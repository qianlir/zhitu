# 上海市嘉定区安亭高级中学 (嘉定区安亭高)

> 上海市嘉定区安亭高级中学地处嘉定国际汽车城中心城镇安亭镇，是安亭镇唯一的全日制公办高中，2010年被命名为区实验性示范性高中。学校前身可追溯到清道光八年（1828年）建造的震川书院，为纪念明代著名散文家归有光在安亭的讲学业绩而建。学校占地90余亩，环境幽雅，现有教学班24个、学生近千名、专任教师百余人，其中正高级教师1人、高级教师22人。以'开明睿智、甫德广才'为校训，践行'让师生共同健康成长'办学理念，以'培养有光的生活者'为育人目标。先后荣膺国家生涯教育示范校、全国校园心理健康教育示范单位等。
