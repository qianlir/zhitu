# 上海中学 (上中)

> 上海中学创办于1865年的龙门书院，当之无愧「四校之首」。清北复交录取率常年领跑全市，进入重点高校比例超过99%。2008年首设科技班（物理/生命/医学/计算机/化学五方向），2012年设工程班（工程/通讯/环境/能源/海洋），是国家「拔尖创新人才培养基地」试点校。2025年清北强基计划录取30人，全市第一。
