# 上海中学 (上中)

> 上海中学创办于1865年龙门书院，四校之首。数学竞赛全国领先：2024年CMO获17金11银，8人入国集（全市最多），邓乐言获全国唯一满分；2023年高联一等奖38人。IMO金牌累计全国第一，2026年IMO将在上中承办。清北复交录取率常年领跑，重点高校比例超99%。设科技班（物理/生命/医学/计算机/化学）和工程班，是国家「拔尖创新人才培养基地」。
