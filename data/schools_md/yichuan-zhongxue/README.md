# 上海市宜川中学 (宜川)

> 上海市宜川中学创建于1957年，1978年被定为普陀区重点中学，2005年正式获评上海市实验性示范性高中。2004年在原址建成现代化高级中学，总投资2.1亿元，建筑面积达4万平方米。学校秉持'生命之舟宜航之川'的办学理念，形成了科创教育（与复旦、交大合作建设飞行创想、人工智能课程）和国际商科教育（与法国里昂商学院合作）的特色品牌，同时以'书院教育''戏剧教育'为亮点。知名校友包括奥运冠军刘翔。连续数年高考本科达线率超过90%。
