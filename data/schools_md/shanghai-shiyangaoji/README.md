# 上海市实验学校 (上实)

> 上海市实验学校是「三公」之一，全市唯一十年一贯制学校，根据学生认知发展节奏实行弹性学制。高中创新班特需课程可自主申报研究项目。2023年清北裸分上线4人（录取8人），特需创新班均分591.7，综评率92.3%。
