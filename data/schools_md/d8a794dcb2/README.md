# 上海七宝德怀特高级中学 (七宝德怀特高)

> 上海七宝德怀特高级中学是由上海市七宝中学和美国纽约市德怀特学校合作举办、上海市教育委员会批准设立、中国教育部备案的上海第一所独立设置的中外合作高中。学校保留中国核心课程（语文、历史、地理、政治）和优势课程（数学），汲取德怀特学校IB课程经验，10年级开设IBDP预备课程，11-12年级开设IBDP课程。2018年经剑桥国际CAIE授权增设A-Level课程。2025学年计划招收10年级学生200人，面向上海、外省市及境外生源。
