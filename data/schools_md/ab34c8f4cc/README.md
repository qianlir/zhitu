# 上海宝山区民办维尚高级中学 (宝山区民办维)

> 上海宝山区民办维尚高级中学于2023年创办，与WLSA上海学校（世界名中学联盟）关系密切，是WLSA宝山校区正式取得高中办学资质后的民办高中。学校以'学习、引领、创新'为校训，以'赋能学生蓬勃发展'为教育理念，坚持五育融合、品质教育，创新适性、多元开放的培养路径。学校为四年制民办高中，可寄宿，各类教育教学设施齐全，配备体育馆、室内高尔夫、篮球场、网球场、五人制足球场、实验室、阅览室、礼堂等。WLSA上海学校曾实现'哈耶普斯麻'大满贯录取，升学成绩优异。学费80000元/学期。
