# 上海市华东模范中学 (华东模范中学)

> 华东模范中学创办于1945年，由中共地下组织领导筹建，著名教育家陈鹤琴任首任校董会主席。1983年复校，2005年被命名为上海市实验性示范性高中。学校以"双通教育"（通识教育+通能教育）改革为核心特色，形成了"团结民主，求真务实，开拓创新，敢为人先"的优良传统。曾培养出乒乓球国手郑敏之、篮球国手丛学娣等体育人才，是上海市体育传统项目（篮球）学校。六次获上海市文明单位称号。
