# 上海青浦区宏润博源高级中学 (青浦区宏润博)

> 上海宏润博源学校成立于2018年，由上市公司宏润建设集团与播文教育联合举办的国际化高中，为希望未来就读国外本科的学生提供全日制国际化教育。学校由中国国际教育先驱、原上海中学国际部主任陈文国先生领衔创办，获得美国百年名校格里格斯学院（GIA）上海独家授权，开设美国高中课程及AP课程。校园位于青浦区朱家角镇课植园路，毗邻古镇旅游区，北朝大淀湖，总用地面积约4.3万平方米。师生比1:4，80%以上教师拥有硕士及以上学位，毕业于普林斯顿、哈佛、清华、牛津等顶尖学府。地铁17号线朱家角站可达。
