# 上海闵行区诺达双语学校 (闵行区诺达双)

> 上海闵行区诺达双语学校（原诺德安达双语学校）自2016年9月建校，是一所融中西文化精粹、推进双语教学的十二年一贯制民办学校。校园总占地面积50亩，位于闵行区华漕镇，毗邻虹桥商务区。课程在严格遵循上海市中小学二期课改大纲的同时融入全球优质教育资源，高中阶段通过IB项目提供出国留学机会。作为Nord Anglia教育集团战略合作伙伴，学生受益于与茱莉亚表演艺术学院、麻省理工学院（MIT）及伦敦大学国王学院（KCL）等的合作项目。为6至18岁学生提供融合中西文化精粹的双语教学。
