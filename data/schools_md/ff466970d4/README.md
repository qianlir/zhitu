# 上海浦东新区民办惠立学校 (浦东新区民办)

> 上海浦东新区民办惠立学校（Hiba Academy Shanghai）位于浦东新区前滩国际商务区，2018年8月创办，由上海陆家嘴集团投资校园建设、惠立教育集团创办并聘请英国惠灵顿公学提供教育咨询服务。学校面向1-12年级学生提供融合中西方教育的双语教育，是IB世界学校、国际学校协会CIS正式会员。高中部实施IGCSE和IBDP课程体系。秉持'惠贤育能，立德树人'理念，致力于在沉浸式双语环境中发展学生双语双文化能力。提供丰富的课外活动、学院制、体育、艺术等校园生活。
