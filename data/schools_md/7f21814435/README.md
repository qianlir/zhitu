# 上海田家炳中学 (田家炳中学)

> 上海田家炳中学创办于1997年，由香港实业家田家炳先生投资兴建，是上海市面向全国和境外招生的民办重点中学。学校占地40余亩，建筑面积2万多平方米，是上海首批民办特色创建学校，荣获'中国社会组织评估等级5A证书'。自2006年起开设国际课程，涵盖美国高中课程+AP及英国A-Level课程。建有创客中心（含机器人训练场、3D打印室、天文室、VR体验区）、室内体育馆、高尔夫练习场等。外籍教师14人，平均教龄超8年。
