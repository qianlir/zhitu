# 上海市风华中学 (风华中学)

> 上海市风华中学前身为1947年创办的私立兴慈中学（后更名粹文中学），1978年被确定为闸北区重点中学，1983年与原风华中学合并冠现名，2015年随区划并入静安区，是静安区实验性示范性高中。学校占地46亩，建筑面积24000平方米，有特级教师2人、高级教师34人、区学科带头人6人，享受国务院特殊津贴2人。校训'学会求知，学会做人'，在自制教具、心理健康教育、教学科学化管理三方面形成特色。获世界卫生组织'健康促进学校金奖'等荣誉。
