# 华东师范大学第二附属中学 (华二)

> 华师大二附中被誉为「竞赛金牌摇篮」，国际奥赛累计25枚金牌居全国前茅，物理和信息学奥赛尤为突出。国际科学与工程大赛获奖20+项全国第一，两位学生获小行星命名殊荣。1978年被确定为上海唯一教育部直属重点中学，现拥有浦东张江和闵行紫竹两个校区。2025年清北强基计划北大18人，合计22人。
