# 上海市民办交大南洋中学 (民办交大南洋)

> 上海市民办交大南洋中学由上海交大教育集团于2000年创办，是一所民办全日制寄宿制高级中学。学校占地面积150亩，建筑面积超5万平方米，配备智能化教学设施及标准化运动场地，施行全寄宿制管理。坚持'只有适合的教育才是最好的教育'办学思想，倡导'承认差异、提供选择、多元培育、终身学习'教育理念。走特色（艺体传媒类）立校的内涵发展之路，近三年高考本科达线率稳定在90%以上。2007年获教育局批准备案国际课程教育资质，2012年成为上海市教委国际课程试点学校。先后获上海市安全文明校园等荣誉。
