# 上海市位育中学 (位育)

> 上海市位育中学创办于1943年，校名取自《中庸》'致中和，天地位焉，万物育焉'，创始人为著名教育家李楚材先生。学校1956年改制为公办，1998年迁址至徐汇区华泾镇位育路1号，2004年通过上海市首批实验性示范性高中评审。江泽民同志亲笔为学校题写校名。校园占地186亩，建筑面积7万余平方米，拥有博育楼、天文台、400米跑道标准足球场（含2000座看台）、室内游泳馆、网球场等10余个不同类型球场。本部设30个教学班，是上海知名的寄宿制高中。
