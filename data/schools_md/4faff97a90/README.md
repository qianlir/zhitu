# 上海宋庆龄学校 (宋庆龄学校)

> 上海宋庆龄学校由中国福利会创办，2008年成立，以纪念宋庆龄女士并践行其大爱精神和教育思想。学校坐落于青浦区赵巷特色居住区，占地近150亩，设有中国部和国际部，实施十二年贯通教育，现有1500余名学生。校训'润己泽人·缔造未来'，办学宗旨为'把最宝贵的东西给予儿童'。教师100%本科以上学历，硕士以上占45%，高中部博士占比53.9%，外籍教师44%。是IB候选学校，与华东师大二附中、上海交大、UCLA等建立合作。学校绿化率60%，配备游泳馆、宋庆龄剧场等一流设施。
