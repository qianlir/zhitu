# 上海师范大学附属嘉定高级中学 (师范大学附属)

> 上海师范大学附属嘉定高级中学前身为1998年创办的嘉定区李园中学（由1904年启良中学高中部发展而来），2006年更名为上海外国语大学嘉定外国语实验高级中学，2010年成为区实验性示范性高中，2021年更名为上海师范大学附属嘉定高级中学。学校占地35007平方米，建筑面积18436平方米，校内汇龙河连通嘉定孔庙泮水和汇龙潭，是沪上距离嘉定孔庙最近的高中。现有30个行政教学班、约1200名学生、专任教师118名。作为全国首所将师范教育前置的高中，学校聚焦'教师教育'特色定位，对接上海师范大学卓越教师培养计划。先后被评为上海市特色普通高中建设项目学校、全国航空航天特色学校等。
