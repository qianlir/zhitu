# 上海市古美高级中学 (古美高级中学)

> 上海市古美高级中学创建于2007年7月，由原基地附中与市西实验中学高中部合并组建而成，是闵行区教育局直属公办普通高级中学。学校以'一切在于人，一切为了人'为办学理念，实施'依法治校、以德立校、科研兴校、教学强校'的办学方略。校训为'修德、笃学、健体、尚美'。学校注重教育信息化建设，构建了以'智慧课堂'为核心的数字化教学模式，毗邻七宝中学，具有良好的区位优势。
