# 复旦大学附属中学青浦分校 (复附青浦)

> 复旦大学附属中学青浦分校创办于2015年，由青浦区人民政府、复旦大学和上海市教育委员会合作举办，是实验性示范性寄宿制高级中学。学校坐落于青浦淀山湖新城，占地163亩，建筑面积7万余平方米，由国际著名建筑师张永和先生设计，四面环水，采用围合式建筑布局。学校实行书院式培养模式与走班制分层教育，每8-10名学生配备1名导师。校训沿用复旦大学'博学而笃志，切问而近思'。现有正高级教师4名，博士4人，硕士45人。
