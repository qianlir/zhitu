# 上海民办包玉刚实验高中 (民办包玉刚实)

> 上海民办包玉刚实验学校建于2007年，由包氏家族为纪念已故船王包玉刚先生对中国教育的贡献而捐资创建的非营利双语国际学校。学校获得CIS（国际学校协会）和WASC（美国西部学校和学院协会）双重认证，提供小学一年级至高中十二年制教育，现有约1600名中外籍学生。高中部位于松江区三新北路，坐落于泰晤士小镇，为英伦风格建筑，提供英美模式的寄宿教育。9年级开设IGCSE课程，10-12年级开设IB大学预科课程（2015年获IBO授权）。学校核心价值观为'仁、义、平'，倡导全人教育，IB平均分达40分，曾有8位满分45分学生。
