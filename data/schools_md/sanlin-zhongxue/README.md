# 上海市三林中学 (三林)

> 上海市三林中学创立于清光绪二十二年（1896年），前身为'三林书院'，由地方教育家秦荣光等集资创办，是一所拥有百年文化底蕴的老校。1953年改为公立，1978年被列为上海县重点中学，1993年划归浦东新区。2009年成为浦东新区首批实验性示范性高中。学校以'勤敬诚肃'为百年校训，形成'仰高'文化：敬仰先哲、追求卓越、自立立人、和谐发展。现有高中教学班30个，在校学生1140人，占地53亩。学校拥有三林刺绣社（非遗传承）、根与芽环保社团等特色社团，先后获评上海市中学生行为规范示范校、市数字化实验学校等。
