# 上海市民办文绮中学 (民办文绮中学)

> 上海市民办文绮中学始创于1947年，1951年并入上海县初级中学（闵行中学前身），1994年恢复建制，是上海市重点高中闵行中学旗下的民办完全中学。学校坚持'以学生发展为本'的办学理念，以'健康、责任、民主、创新'为校训，注重'勤奋、严谨、求实、创新'校风。拥有2万多平方米现代化教学大楼，配备理化实验室、机器人实验室、计算机房、音乐室、语音室、地质馆等21个专用教室及图书馆、游泳馆、健身房等设施。现有学生1700余人，教职工260余人，中高级教师占75%以上。文绮汇点学校（2016年成立）为其国际部，由文绮中学与美国加州汇点高中合作创办。
