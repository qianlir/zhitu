# 上海工商外国语职业学院附属中学 (工商外国语职业学)

> 民办上海工商外国语职业学院附属中学创办于2001年9月，位于浦东新区南汇大学城内，是由上海新世纪教育发展有限公司出资举办的全日制寄宿制完全中学。学校占地28400平方米，建筑面积近18845平方米，依托上海工商外国语职业学院的教学资源优势，以外语教育为特色。设有出国班和普通班两种培养方向，已与日本梅花女子大学、日本东海大学、韩国釜山外国语大学等多所海外院校建立友好合作关系。目前拥有高一至高三共14个教学班，在校学生545人，教职工50人，其中专职教师33人。
