# 上海市大同中学 (大同)

> 大同中学由胡敦复先生于1912年创办，培养了39位两院院士，钱其琛、傅雷等知名校友。英语和化学是传统强项。1987年率先在全市开展高中课程整体改革，是课改先驱。一本率85%以上，2010年引进PGA全球通用证书项目国际课程班。近年成绩优于格致，黄浦区另一块招牌。
