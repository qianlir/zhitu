# 上海市市北中学 (市北)

> 市北中学由唐乃康先生于1915年创办，孙中山先生题词「作育人才」，蔡元培先生题「诲人不倦」。学校是首批市实验性示范性高中，现有30个班约1000名学生，以科技教育和数学竞赛见长。
