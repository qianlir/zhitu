# 上海市松江区第四中学 (松江区第四中)

> 上海市松江区第四中学（松江四中）创办于1952年10月，位于古镇泗泾，占地72亩。学校以'尚美求真，明德问学'为校训，秉承'让美成就未来'的办学理念。形成'多规格发展，多层次成才'的育人渠道，在美术和体育方面取得丰硕成果，同时积极拓展播音、主持、编导等新领域。学校拥有根艺馆、学生艺术作品展示馆、安全体验教室等特色设施。先后获评上海市文明校园、市'五一劳动奖状'、市德育先进集体等荣誉。
