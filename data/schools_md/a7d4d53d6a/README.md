# 上海市民办民远中学 (民办民远中学)

> 上海市民办民远高级中学创建于1996年，由华东师范大学等高校的教育专家和浦东新区重点中学退休高级教师共同参与创办的全日制民办高级中学（现已更名为上海市浦东新区民办浦实高级中学）。学校实行董事会领导下的校长负责制，每学期聘请市区重点中学退休和在职高级教师任教，并邀请华东师大等高校专家教授兼课，英语口语等选修课聘请外籍教师任教。学校注重德育品行教育，通过爱心扶贫、环保行动、学生社团、拓展训练等活动培养学生感恩意识和社会责任感。
