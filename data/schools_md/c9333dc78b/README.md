# 上海松江区爱菊学校 (松江区爱菊学)

> 上海松江区爱菊学校是爱菊教育继徐汇、青浦之后在松江创办的又一所高端民办学校，覆盖高中、初中、小学的十二年一贯制民办学校。占地面积约102亩，建筑面积85537平方米，总投资约10亿元，2023年9月借址开学，新校舍将于2025年9月建成启用。秉承爱菊教育'爱知博艺，菊雅竞芳'办学理念，以'笃志明理，躬行求真'学校文化为指引。围绕核心素养开齐开足国家课程，以人文素养、国际视野、数理思维、信息科技、体育、艺术、社会实践为课程体系。高中用A-Level与WACE课程解锁世界高校机会。由爱国实业家刘浩清先生创办的爱菊教育集团举办。
