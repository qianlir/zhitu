# 上海市文来中学 (文来中学)

> 上海市文来中学由七宝中学于1994年创办，校名取自'博文约礼，鉴往知来'。2004年高中部整体搬迁至古美社区虹莘路校区，是七宝中学教育集团核心学校、闵行区首批实验性示范性高中。学校以'人文立校、多元成才'为办学理念，形成'多元化教育、多课程选择、多渠道成才'的办学特色。2006年引进国际课程，设有中美班、中加班、中日班。2014年获评上海市二十一家国际课程试点校之一，2017年成为上海特色普通高中项目校。高考一本率曾达32%，本科率89%。
