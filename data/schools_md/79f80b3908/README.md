# 上海音乐学院附属安师实验中学 (音乐学院附属安师)

> 上海音乐学院附属安师实验中学（简称上音安师）坐落于苏州河桥畔，历史可追溯至1865年的龙门书院，是上海教育历史发展文脉的重要组成部分。2003年5月由普陀区政府和上海音乐学院合作共建'音乐美育'特色学校，2020年被市教委评为'上海市特色普通高中'，是上海市唯一一所音乐美育特色高中。学校设有独立的音乐教学楼及音乐多媒体实验楼，以'和乐教育'为核心文化体系，'爱音乐、爱智慧'为办学理念，培养'人格健全、气质优雅'的学生。前身历经龙门师范学堂、江苏省立第二师范学校、上海市安亭师范学校等阶段。
