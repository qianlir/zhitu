# 上海市长征中学 (长征中学)

> 长征中学创办于1965年，是普陀区实验性示范性高中。2019年与上海市真如中学合并，现有52个教学班、在校学生超过2000人。学校以计算机和英语口语为两大特色，围绕"为面向未来工程匠师奠基"的培养目标，打造设计奠基、专业体验、品格养育、综合实践四大课程群。连续七届被评为上海市文明单位，是首批上海市行为规范示范学校。从1998年起坚持聘请英美澳外籍教师教授英语口语。
