# 上海市同洲模范学校 (同洲模范学校)

> 上海市同洲模范学校成立于1998年，是一所高标准、新模式、现代化的民办十二年一贯制学校，2019年加入同济大学存志学校集团。名誉校长潘多是世界上第一位从北坡登上珠穆朗玛峰的女登山家，学校以'塑造学生的攀登精神'为办学理念。校训'同心、同力、同创、同达'。现有56个班级、2200余名学生。学校重点培养艺术表现能力、体育竞技能力、科技创新能力和语言交际能力四种能力，建有攀岩壁、击剑馆、围棋馆、手球馆等特色设施。先后荣获教育部中华优秀文化艺术传承学校、国家奥运后备人才培训基地、上海市艺术教育特色示范学校等荣誉，是宝山区唯一连续三轮荣获上海市民办特色校的学校。
