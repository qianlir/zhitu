# 上海市行知实验中学 (行知实验中学)

> 上海市行知实验中学是在伟大的人民教育家陶行知先生亲手创办的原育才学校旧址上于1993年重建的一所完全中学。学校地处宝山区大场镇，占地面积39123平方米，建筑面积18324平方米，绿化覆盖面积达43%。现有初高中35个教学班约1500人，教职工130人，中高级职称教师占64%。校址留存的原育才学校校舍'行知育才旧院'为宝山区区级文物。学校是宝山区实验性示范性高中、上海市特色普通高中建设项目学校、上海市中学生行为规范示范校、中华优秀传统文化研习暨上海市非遗进校园优秀传习基地。高考升学率连续9年达90%以上。
