# 上海市曹杨第二中学 (曹杨二中)

> 曹杨二中创建于1954年，是普陀区唯一的市实验性示范性高中，被称为2024届「最大黑马」。德语班和博雅班是招牌特色，现有37个教学班1600余学生。学校是联合国教科文组织ESD示范校，国际化办学特色鲜明。
