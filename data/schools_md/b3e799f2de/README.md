# 上海市浦东新区民办浦实高级中学 (浦东新区民办)

> 上海市浦东新区民办浦实高级中学（原上海市民办民远高级中学）创建于1996年，由华东师范大学等高校教育专家和浦东新区重点中学退休高级教师共同创办的全日制民办高级中学。学校实行董事会领导下的校长负责制，重视素质教育，实施必修课、选修课和活动课'三个板块'课程结构。每学期聘请市区重点中学退休和在职高级教师任教，同时聘请高等院校专家教授兼课。创办者还筹建了上海民远学院，为毕业生提供更多大学深造机会。学校面向全体学生，注重德智体美劳全面发展。
