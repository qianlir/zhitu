# 上海市民星中学 (民星中学)

> 民星中学创建于1995年，坐落于杨浦区中原地区，是一所公立高级中学。学校围绕"向美而行，以美育人"的办学理念，以美育为突破方向构建"一体两翼"办学特色：以信息化办学为主体，以科技教育和影视教育为双翼。是联合国EPD（环境、人口与可持续发展教育）实验学校，杨浦区科技教育特色学校。多次获得上海市文明校园称号，是上海市花园单位。
