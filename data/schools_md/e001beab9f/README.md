# 上海浦东新区民办万科学校 (浦东新区民办)

> 上海浦东新区民办万科学校于2017年9月建校，是万科德英乐教育旗下十二年一贯制民办学校。学校位于浦东康桥地区，占地85亩，校园绿化面积达72%。学校是IBO小学项目PYP、中学项目MYP、大学预科项目DP和职业教育项目CP的授权学校，同时获得AP授权和Cognia认证。高中部实行全寄宿制，开设IGCSE、IBDP、AP等学术课程。学校秉承万科精神，坚持双语特色，坚持中国基础教育优势与国外教育精华有效结合，以培养立身世界的中国人为目标。设有多条校车路线，交通便利。
