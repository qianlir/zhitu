# 上海市西郊学校 (西郊学校)

> 上海市西郊学校创办于2004年，前身可追溯至1868年法华镇的法华义塾，是长宁区一所集小学、初中、高中的十二年一贯制公办学校。地处虹桥临空园区核心区域，毗邻上海动物园，占地面积18100平方米，建筑面积19751平方米，现有42个教学班、学生1400人左右、教职工150余人。以'体教结合'为办学特色，校训'正行精进'。是教育部100所创建特色高中学校之一，获'上海市体教结合先进单位'等荣誉。建有体操、蹦床、柔道等专业训练场馆。
