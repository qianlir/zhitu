# 上海大学附属中学 (上大附中)

> 上海大学附属中学创办于2003年，前身可追溯至1923年成立的上海大学中学部，是上海市实验性示范性寄宿制公立高中。已故著名科学家钱伟长院士为学校首任名誉校长并亲笔题写校名。学校位于宝山区，毗邻上海大学，占地210亩，建筑面积7万余平方米。现有38个教学班（含新疆部），在校学生1434人。设有致远班、伟长班、创新班三类特色实验班。校训为'明德·致远'，办学思想为'创新进取·和谐幸福'。配备4000平方米创新活动中心和国际标准运动场馆。先后获评全国十佳创新学校、上海市戏剧特色学校等。
