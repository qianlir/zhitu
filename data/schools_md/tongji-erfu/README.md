# 同济大学第二附属中学 (同济二附)

> 同济大学第二附属中学由同济大学和普陀区政府于2002年合作创办，在撤并原普陀中学和长寿中学基础上筹建，2003年正式挂牌。2007年被评为普陀区实验性示范性高中，2019年被命名为上海市特色普通高中。学校占地58亩，分南北两校区，现有39个教学班、1300余名学生。秉承'同舟共济、自强不息'精神，以理工特色见长，建有3S技术、地球探测、生命科学等八大创新实验室，是联合国教科文组织中国可持续发展项目示范校。
