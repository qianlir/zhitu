# 上海市徐汇中学 (徐汇)

> 上海市徐汇中学创办于1850年（始名徐汇公学），由天主教传教士南格禄创办，是中国近代最早按西方办学模式创立的学校之一，有'西学东渐第一校'之美誉。学校首开西洋班级化教学，是中国近代教育史上第一所开设数理化、法语英语拉丁语、音体美课程并拥有实验室的学校。2005年评为徐汇区实验性示范性高中，2020年评为上海市科创特色高中。校训为'汇学'，以'成己达人，兴学兴邦'为文化内核。培养杰出校友六万余人，包括马相伯、翁文灏、张充仁、傅雷、褚君浩院士等。现有3个校区、79个班级、3420多名学生。
