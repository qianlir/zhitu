# 上海市宝山华曜高级中学 (宝山华曜高级)

> 上海市宝山华曜高级中学是华东师范大学基础教育集团、华师大二附中基础教育集团核心成员校，2023年获得办学许可正式创办。学校秉承并发展华师大二附中办学理念，以'追求卓越、培养创造未来的人'为育人目标，坚持精品化课程、精细化服务、精英化培养。由宝山区教育局直接领导、华东师大二附中业务指导。学校是英国切斯文法学校和日本行知学园在上海的重要战略合作伙伴，提供国内高考和国际升学（英国、日本方向）多通道升学路径。首届高一招收76人，并设有丰厚奖学金政策。
