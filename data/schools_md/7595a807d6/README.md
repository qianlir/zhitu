# 上海市北虹高级中学 (北虹高级中学)

> 上海市北虹高级中学前身为1874年（清同治十三年）法国天主耶稣会创办的圣芳济学院（St.Francis Xavier's College），1884年迁至虹口南浔路现址，是一所拥有150年历史的全日制公办高中。现为虹口区实验性示范性高中、上海市中小学课程教材改革研究基地。2016年被市教委列为戏剧特色学校，2020年确定为虹口区戏剧龙头学校。2024年因与上海音乐学院合作办学，更名为'上海音乐学院虹口区北虹高级中学'。学校占地逾9000平方米，有19个教学班，建有北虹剧场、地下篮球馆等设施。
