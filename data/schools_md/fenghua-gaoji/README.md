# 上海市奉贤区奉城高级中学 (奉城高级)

> 上海市奉贤区奉城高级中学前身为创办于1978年7月的奉城中学，2001年撤消初中部改为全日制普通高级中学并更名。2006年秋搬迁至奉城兰博大街规划区，占地110亩，校舍为西班牙风格，全体学生住校。学校以'自律求真、活学求新'为办学理念，实施'自主教育、多元发展'特色教育模式。是上海市高中课改研究基地学校、奉贤区学校管理等级达标唯一的B1级学校。2010年成为全国33所入围'教育部中央教科所特色高中'评选的申报学校之一。
