# 上海市西南位育中学 (西南位育)

> 上海市西南位育中学创建于1993年，是上海市最早创办的公立转制完全中学之一。2005年正式转制为民办学校，2022年起由徐汇区教育基金会主办。2007年评为徐汇区实验性示范性高中。学校拥有三个校区（西校区宜山路、东校区小木桥路、北校区桂林路）、79个班级、三千余名师生。以'中和位育'和'渐进积累'为核心思想，形成四大特色项目：中华传统文化德育、双本双语双外语教学、机器人与信息技术科创、男篮女排高雅艺体。校训为'追求完美，力求发展'。获评全国国防教育示范校、全国排球特色学校等。
