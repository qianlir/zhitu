# 上海市复旦中学 (复旦中学)

> 上海市复旦中学前身为我国著名教育家马相伯创办于1905年的复旦公学，与复旦大学同根同源。百年历程中先后经历'复旦公学、复旦大学附属中学和复旦中学'三个阶段。孙中山先生曾任学校首席校董，严复、于佑任、邵力子等为创校先贤。1956年改为公立学校，2015年被命名为上海市实验性示范性高中。学校设有东校区（华山路）和西校区（淞虹路），由长宁区人民政府和复旦大学共建。校训为'博学而笃志，切问而近思'，以'博雅教育'和创新素养培育为办学特色。培养了陈寅恪、竺可桢等文化名人及15位院士。
