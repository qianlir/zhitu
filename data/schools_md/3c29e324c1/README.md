# 上海师范大学附属宝山罗店中学 (师范大学附属)

> 上海师范大学附属宝山罗店中学（原上海市罗店中学）坐落于有700年历史的罗店古镇，创立于1942年，是宝山区实验性示范性寄宿制高中，2022年被评为上海市特色普通高中。学校占地150亩，建筑面积6.3万平方米，建有11栋高标准楼群及400米标准跑道。现有37个教学班，学生近1500人，教职工143人。2007年与上海师范大学合作共建。学校以'艺术见长，尚美成人'为办学思想，管乐团建于1992年，被命名为'上海市学生艺术团罗店中学管乐团'，现有团员250余名，曾在上海大剧院等开设专场音乐会。
