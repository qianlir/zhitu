# 上海市延安中学 (延安)

> 延安中学建于1946年，1979年邓小平同志亲笔题写校名。实行全寄宿制管理，理科尚有特色，竞赛师资雄厚（特级教师7名，高级教师70+名）。设延智/延创/延行/文心四类实验班。一本率84%，长宁区成绩一枝独秀。全市排名约20名左右。
