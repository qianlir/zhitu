# 上海市崇明区民本中学 (民本)

> 上海市崇明区民本中学创办于1925年，校名源自《尚书·五子之歌》'民惟邦本，本固邦宁'及《孟子》民本思想，是一所拥有近百年历史的区重点高级中学。学校以'厚德·笃行'为校训，秉承'以人为本，协同发展'的办学理念，构建家庭、学校、社会'三位一体'协同育人机制。1946年建立中共地下党支部，红色基因传承深厚。校园占地43281平方米，设21个教学班。高考上线率稳定在98%以上，本科率保持在75%以上。先后被评为全国教育科研先进集体、全国青少年篮球特色学校、上海市文明单位等。
