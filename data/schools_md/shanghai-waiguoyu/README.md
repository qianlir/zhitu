# 上海外国语大学附属外国语学校 (上外附中)

> 上海外国语大学附属外国语学校（上外附中）创办于1963年，是周恩来总理亲自批示成立的全国首批七所外国语学校之一，直属国家教育部，被誉为'培养外语外交人才的摇篮'。学校为初高中七年一贯制完全中学，是上海市实验性示范性高中、全国外国语学校工作研究会会长单位。主校区占地33亩，设31个班级、1600多名学生。培养目标为'全面发展、外语见长、文理并举'的国际型预备英才，知名校友包括杨洁篪、王光亚、崔天凯等外交官。
