# 上海大学市北附属中学 (大学市北附属)

> 上海大学市北附属中学创办于1954年，前身为上海铁路第一中学，2000年由上海铁路局整建制移交闸北区人民政府，2002年与育群中学及和田中学高中部合并，更名为上海大学市北附属中学。学校以'三格教育'思想推进教学改革，多年来高考升学率高位稳定，以'普通中学生源、重点中学质量'著称，被评为区办学先进单位。1998年及2000年新建两幢教学大楼，建筑面积达12000平方米，设有校园网络、实验室、图书馆、电子阅览室等现代化设施。
