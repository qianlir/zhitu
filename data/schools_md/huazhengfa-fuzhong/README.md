# 华东政法大学附属中学 (华政附)

> 华东政法大学附属中学（华政附中）前身为创办于1954年的上海市番禺中学，首任校长任命书由陈毅市长签发。2009年与华东政法大学联合办学更名，成为全国第一所政法类高校附中。2018年获评上海市特色普通高中。学校秉承'明德精业'校训，融入'尚法'精神，构建'明德、尚法、精业'三大类百余门校本课程。建有尚法创新实验楼，设有模拟法庭、模拟听证等特色设施。学校占地约14000平方米，现有在校学生629人。先后获评全国特色学校、上海市文明校园、全国普法与依法治理工作联系点。
