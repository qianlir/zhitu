# 上海外国语大学附属大境中学 (外国语大学附)

> 上海外国语大学附属大境中学创办于1962年，1979年被命名为市重点中学，2005年成为首批上海市实验性示范性高中。自2000年起与上海外国语大学合作办学，以外语教学为核心特色，英语实施小班化教学，德语教学已有十余年历史，是上海市较早开展双语教学和第二外语教学的学校。学校拥有18层教学大楼、五百座位大礼堂、5500平方米草坪球场及射击中心，是全国外语教学实验学校、全国现代教育技术实验学校、中国创造学会创造教育优秀实验基地。
