# 上海市育才中学 (育才)

> 育才中学由犹太慈善家嘉道理于1901年创办，1953年首批上海市重点中学，著名教育家段力佩曾任校长。1997年迁至嘉定马陆，全寄宿制。首批市实验性示范性高中。
