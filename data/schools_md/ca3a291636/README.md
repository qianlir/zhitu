# 上海市通河中学 134002 宝山区 公办 区实验性示范性高中144 上海市顾村中学 (通河中学 1)

> 上海市通河中学创建于1991年，是宝山区首批实验性示范性高中、上海大学基础教育集团项目学校。学校占地约43亩，分为行政、教学、运动、特色、生活五大区域，形成'知明''知新''知行''知微''知足'五大楼系。学校围绕'多元发展，健康成长'办学理念，以科技、美术、体育为三大特色课程。先后荣获'全国首批科学教育实践基地''市科技特色示范校''市行为规范示范校'等称号。高考本科达线率连年创历史新高。
