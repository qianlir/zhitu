# 上海市同济中学 (同济中学)

> 同济中学前身为1907年德国医学博士埃里希·宝隆创办的同济医学堂德文科，是杨浦区历史最悠久的学校。1924年改为同济大学附属中学，1950年定名上海市同济中学，1959年定为上海市重点中学。学校以德语教学为特色，2003年恢复德语课程并与德国高中建立友好合作。享有"院士摇篮"美誉，诞生了吴孟超等12位两院院士。篮球是传统强项，高中男篮在全市名列前茅。现为杨浦区实验性示范性高中。
