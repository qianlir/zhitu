# 上海市控江中学 (控江)

> 控江中学1953年建校即被列为重点学校，杨浦区绝对的区内第一，裸考录取分常年位于全市前列。人文氛围排名全市第三，创新教育有特色。杨浦区仅3所区属市重点，另两所均无法挑战控江地位，是老牌名校中保持稳定的代表。
