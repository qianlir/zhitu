# 上海闵行田园高级中学 (闵行田园高级)

> 北京外国语大学附属上海闵行田园高级中学（简称北外田园）创建于2003年9月，由创立于1948年的颛桥中学高中部分立而成。学校是闵行区实验性示范性公办高中、七宝中学教育集团核心学校、颛桥学区化办学领衔学校。占地25亩，建筑面积1.3万平方米，办学规模18个教学班约700人。教职工105名，其中高级教师32人。学校在'美育引领，和谐发展'办学理念下，以艺术、体育、外语、写作为特色，是上海市文明校园、上海市特色高中建设项目学校、全国校园文学示范校、全国中小学中华优秀传统文化传承学校。
