# 上海理工大学附属中学 (理工大学附属)

> 上海理工大学附属中学创建于1957年，2003年依托上海理工大学合作办学更为现名。学校在'尚理'办学理念引领下，定位'工程素养'培育特色，2019年被上海市教委命名为上海市特色普通高中。现有24个班级、730名学生，拥有特级教师2名、中高级教师86人占全校教师86.9%。是上海市科技教育特色示范学校、首批'中小学人工智能教育实验校'（2019年全国首批）。形成了计算机、机器人、头脑OM三大科技教育品牌。学校是上海市行为规范示范校、上海市体育传统项目学校。
