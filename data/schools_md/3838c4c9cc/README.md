# 上海师范大学第四附属中学 (师范大学第四)

> 上海师范大学第四附属中学是上海市教委直属学校，位于奉贤区海湾旅游区，东邻临港新片区，南枕杭州湾，西接上海师范大学（奉贤校区）、华东理工大学和上海应用技术大学。学校依托上海师范大学教育资源，是上海师范大学在奉贤区的附属学校之一。学校办学环境优越，地处大学城区域，拥有丰富的高校资源和良好的学术氛围，为学生提供多元发展通道。
