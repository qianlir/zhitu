# 上海市崇明区堡镇中学 (崇明区堡镇中)

> 上海市崇明区堡镇中学创建于1992年6月，1998年与创办于1946年的北堡中学合并组成新的堡镇中学，是崇明岛规模最大的完全中学之一。2009年实行初高中分离，现为崇明区实验性示范性高级中学。学校占地53333平方米，拥有400米标准型塑胶跑道田径运动场等完善设施。学校以'高质量、创特色、争一流'为办学目标，坚持全面推进以德育为核心、创新精神和实践能力为重点的素质教育。
