# 上海立信会计金融学院附属高行中学 (立信会计金融学院)

> 上海立信会计金融学院附属高行中学（原上海市高行中学）始建于1947年，是浦东新区十四所老校、名校之一，2010年6月被评为浦东新区实验性示范性高中。2024年3月与上海立信会计金融学院合作办学后正式更名。学校占地56亩，建筑面积17000多平方米，紧邻外高桥自贸区。现有教职员工162名，其中正高级教师1名、高级教师18名、区学科带头人10名。高考本科录取率稳定在85%左右，2018年创纪录达94.2%。是上海市花园单位、上海市依法治校示范校、上海市绿色学校、浦东新区高中'双新'实验校。已故著名作家姚雪垠曾在校任教。
