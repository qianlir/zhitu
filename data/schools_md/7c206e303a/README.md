# 上海市民办尚德实验学校 (民办尚德实验)

> 上海市民办尚德实验学校创建于2003年9月，是一所集幼儿园、小学、初中、高中15年完整教育生态链的寄宿制学校。校园占地216亩，建筑面积14.7万平方米，在校学生7000余名。学校坐落于浦东康桥，2018年开设航头校区。先后获得IB PYP、MYP、DP所有认证，2014年成为首批上海市21所国际课程试点校之一，并开设A-Level和美加课程方向。融合部硕士及以上学历占比80%，师生比1:6，拥有31位IB官方考官。82%融合部学子被世界排名前50大学录取，96%高中部学子考入心仪本科院校。特色社团包括模联、商业联盟等。
