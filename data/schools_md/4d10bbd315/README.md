# 上海市复旦实验中学 (复旦实验中学)

> 复旦实验中学创建于2000年9月，是复旦大学与杨浦区政府合作创办的区属完全中学。其历史可追溯至1925年郭任远创办的复旦大学心理学院附属实验中学。学校是复旦大学基础教育理事单位、复旦附中教育联盟成员校、复旦实验中学教育集团核心校。以中华优秀传统文化教育培育人文素养为特色，2017年起招收复旦附中联盟班。是全国国防教育示范校、上海市中小学心理健康教育示范校。现为杨浦区实验性示范性高中。
