# 上海师范大学附属中学 (上师大附中)

> 上海师范大学附属中学创建于1958年，是市教委直属的五所高级中学之一。2014年总校迁至浦东新校区。浦东总校24个班近1000名学生，另有闵行分校和宝山分校。
