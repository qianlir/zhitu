# 上海市七宝中学 (七宝)

> 七宝中学被称为「第五校」，八大中综合升学实力最强。特级教师15人居全市前列，复交综评录取率超24%。2020年清北复交总录取152人，校内牛娃云集竞争激烈。前身为1947年创办的南洋模范中学七宝分校，2005年成为首批市实验性示范性高中。
