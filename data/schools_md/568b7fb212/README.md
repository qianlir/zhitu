# 上海市第二体育运动学校 (第二体育运动学校)

> 上海市第二体育运动学校（上海市体育中学）创建于1987年，是一所致力培养德智体美劳全面发展优秀体育后备人才的体育类中等专业学校，隶属于上海市体育局和上海市教委。学校占地140亩，位于闵行区莘庄镇，是上海市市级花园单位，连续5个周期获评'国家高水平体育后备人才基地'。拥有400米塑胶标准田径场、亚洲一流200米室内田径馆、国际标准自行车赛车场等设施。作为田径110米栏奥运冠军刘翔和自行车奥运冠军钟天使的母校，是名副其实的冠军摇篮。设有田径、自行车、举重等项目。
