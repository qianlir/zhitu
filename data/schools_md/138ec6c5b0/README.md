# 上海市民办风范中学 (民办风范中学)

> 上海市民办风范中学创建于1995年，坐落于静安大宁地区，现有初高中学生960多人、23个教学班，是全国首批优秀民办学校之一。学校实行董事会领导下的校长负责制，以'让每一个生命得到充分发展'为办学理念，以'风范教育'为特色，培育学生'做一个有风范的中国人'。全校143名教职员工，高级教师占1/3，中高级教师占83%。自主构建'多选择、有分层、扬个性、高品质'校本课程体系。获评上海市民办特色创建校、上海市中小学行为规范示范校等荣誉。
