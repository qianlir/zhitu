# 上海市松江第二中学 (松江二中)

> 松江二中发祥于清乾隆十八年（1753年）的云间书院，现为首批上海市实验性示范性高中。占地10.29万㎡，以宋代遗址云间第一楼为校门。与复旦、华师大等高校合作开展拔尖创新人才培养。近年13人考入清北，生物奥赛获1金13银12铜。校友包括丰子恺、郎静山。
