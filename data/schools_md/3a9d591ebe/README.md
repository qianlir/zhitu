# 上海市松江区科德高级中学 (松江区科德高)

> 上海市松江区科德高级中学（原上海阿德科特学校）是英国阿德科特学校的第一家海外分校，2018年9月正式开学。学校坐落于松江佘山脚下，占地面积30亩，为非营利性民办高中。教职员工70余人，中外教比例3:1，85%以上拥有硕士学位、5位博士学历。开设IGCSE和A-Level课程，每班25人小班制。建校以来三年获得12枚牛剑Offer，超过半数学生进入世界Top50大学。拥有图书馆、多功能厅、足球场、攀岩墙、高尔夫平台、STEAM创新实验室等设施。校名'科德'凝练自'科学求真，明德求善'。
