# 上海市嘉定区中光高级中学 (嘉定区中光高)

> 上海市嘉定区中光高级中学前身中光中学创办于1945年，由周永昌、潘指行等娄塘贤达创办于古镇娄塘，取名'中光'寄寓'中国光复''中兴之光'的爱国深意。1956年改为公立，2005年恢复中光中学校名，2006年高中部整体搬迁至嘉定中心城区，成为公办寄宿制高中。学校占地53亩，现有18个教学班、650余名学生、教职工74人，中高级教师达85%。传承'淳朴、踏实、敬业、奉献'优良传统，确立以文立身、以文益智、以文孕美的'文化立校'办学理念。拥有数字化实验中心、信息化教育中心等设施。荣获全国中小学心理健康教育特色校、上海市书法实验创新校等称号。
