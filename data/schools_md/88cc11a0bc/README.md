# 上海市第五十四中学 (第五十四中学)

> 上海市第五十四中学创办于1955年，是上海市人民政府兴办的第一批公办完全中学之一，2006年被命名为徐汇区实验性示范性学校。学校坐落于号称上海'中南海'的康平路，校址原为民国时期著名园艺家黄岳渊设计的黄家花园。现有初中16个班、高中19个班、学生1200余人，占地16.5亩。以'弘爱求实，自能发展'为办学理念，篮球、民乐、信息技术为特色项目，篮球队在上海高级中学中久负盛誉。是国家级规范汉字书写教育特色校。
