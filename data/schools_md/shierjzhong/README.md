# 上海市第二中学 (市二)

> 上海市第二中学始建于1902年，初为「务本女塾」，是近代中国人创办最早的女校之一。1954年首批上海市重点中学，2005年市实验性示范性高中。科技特色示范校、艺术特色示范校、体育传统项目学校。
