# 上海市沪新中学 (沪新中学)

> 上海市沪新中学创办于二十世纪七十年代后期，是一所在社区内享有良好声誉的公办完全中学。学校占地面积3万余平方米，建筑面积2万余平方米，现有初中四个年级12个班级和高中三个年级12个班级，在校学生1550余人，教职工114人。学校以'居敬持志'为校训，'让学校成为生命活动的和谐家园'为办学目标。2006年被列为浦东新区生命教育试点校，2008年被列为浦东新区'两纲'示范基地学校。学校以陶艺教育和管乐教育为特色，是上海市陶艺中心浦东分中心、联合国教科文组织EPD实验学校。
