# 上海市向东中学 (向东中学)

> 上海市向东中学始创于1912年，前身为上海首所女子师范学校——南洋女子师范学校，由爱国实业家凌铭之先生创办，1927年改名南洋女子中学，1966年正式定名向东中学。学校曾是中共地下党活动基地，享有'红色堡垒'之誉，是一所拥有百年历史和光荣革命传统的公办完全中学。校训'庄敬'，办学理念'庄敬向东，好学向上'。占地面积5171平方米，90%以上教师具有中高级职称。围棋教学特色突出，曾培养出世界业余围棋总冠军刘钧等优秀选手。
