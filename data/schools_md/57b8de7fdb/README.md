# 上海市第十中学 (第十中学)

> 上海市第十中学创办于1906年，前身为爱国民主人士苏本喦女士创办的「民立女子中学」，是上海为数不多由中国人自己出资创办的百年名校。1960年曾被评为上海市重点中学，2011年成为黄浦区实验性示范性高中。学校确立「做更好的自己」办学理念，形成「理科见长、体艺双飞」办学特色，物理化学竞赛屡获佳绩，三棋和版画特色鲜明。知名校友包括著名教育家于漪、中联部原部长李淑铮、女高音歌唱家高芝兰等。
