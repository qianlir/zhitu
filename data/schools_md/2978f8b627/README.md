# 上海市崇明区城桥中学 (崇明区城桥中)

> 上海市崇明区城桥中学创建于1978年，是崇明岛上一所公办高级中学，2009年被评为崇明区实验性示范性高中，2021年被评为上海市特色普通高中。学校面向全市招生，现有24个教学班，全校在职教职工151人，其中高级教师51名。学校秉承'发扬人的优点、发掘人的潜能、发挥人的才干、发展人的特长'教育思想，以尊重和发展学生个性特长为基本思想，形成鲜明的艺术、科技、体育办学特色。先后荣获全国第一批中华优秀文化传承学校、国家级体育传统项目学校、上海市文明单位、上海市艺术教育特色学校、上海市头脑奥林匹克特色学校等荣誉。
