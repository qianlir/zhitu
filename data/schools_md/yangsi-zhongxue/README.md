# 上海市杨思高级中学 (杨思)

> 上海市杨思高级中学创办于1997年，由原杨思中学、杨思高级中学及杨思南校于2000年合并组建，2008年初中部脱钩后成为独立高中。2009年获评首批浦东新区实验性示范性高中。学校以'扬善慎思'为校训，以'心理健康教育'为办学特色，建有功能完备的心理专用教室、心理咨询室、心理宣泄室。校园占地65亩，现有三个年级共24个班级，近千名学生。高级教师占比30%。先后获得全国青少年校园足球特色学校、上海市绿色学校、浦东新区科技特色学校、浦东新区影视教育特色基地等称号。
