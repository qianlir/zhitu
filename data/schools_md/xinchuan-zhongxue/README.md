# 上海市新川中学 (新川)

> 上海市新川中学创办于2010年9月，由原侨光中学高中部与五三中学高中部合并组建，位于浦东新区川沙新镇。2009年5月被正式命名为首批浦东新区实验性示范性高中。2023年秋季启用妙川路新校区。学校以'为了每一位学生充分发展，实施精致教育'为办学思想，校训为'真、实、细'。共开设36门校本拓展课、54个学生研究课题、18个学生社团，涉及六大领域。武术、围棋、书画、合唱是学校的特色项目。现有近千名学生，在职教师86位。
