# 上海市第六十中学 (六十中学)

> 上海市第六十中学前身为1956年私立肇和中学，1963年确定为市重点中学，2007年市实验性示范性高中。信息技术教育特色鲜明，建有千兆校园网。「基础加特长」育人模式。
