# 上海师范大学附属外国语中学 (师范大学附属)

> 上海师范大学附属外国语中学建校于1927年，地处松江老城区，占地106亩，建设用地约6.7万平方米，是一所人文底蕴深厚的现代化学校。学校是松江区实验性示范性高中、上海市特色高中创建学校、上海市科技教育特色学校、国际生态学校、卓越教师培养实践教学基地。1998年交由上海师范大学管理并更为现名，2007年属地化划归松江区领导。现有1662名学生、40个班级、教职工167名。学校拥有完善的教学设施，环境幽雅。
