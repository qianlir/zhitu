# 上海市中原中学 (中原中学)

> 上海市中原中学创办于1994年8月，前身为上海市控江中学中原分校，1995年被上海市教委批准为杨浦区重点中学，现为杨浦区实验性示范性高中。学校位于殷行街道，建筑面积14000平方米，在校学生700余名、教师80余名。2011年加入'上海交通大学附属中学教育联盟'。学校以'和谐发展，追求卓越'为办学精神，拥有4个市级创新实验室、2个区级创新实验室。是上海市花园单位、上海市心理辅导实验学校、上海市体育传统项目学校、上海创造教育实验基地。
