# 上海市崇明区横沙中学 (崇明区横沙中)

> 上海市崇明区横沙中学创建于1958年，是横沙岛上唯一的一所公办完全中学（含初中和高中部），占地35亩，现有11个教学班、305名学生和68名教师。学校位于横沙乡民东路1503号，以'生长课题'为主线推进课题研究，培养学生合理的人生为宗旨。曾获上海市军民共建先进集体、科技教育特色学校等荣誉。2023年接受区教育综合督导，持续推进城乡教育一体化建设。作为海岛学校，承担着为横沙岛居民提供完整基础教育的重要使命。
