# 上海交通大学附属闵行实验学校 (交通大学附属)

> 上海交通大学附属闵行实验学校由闵行区人民政府和上海交通大学合作举办，2024年首届招生，是高起点、高规格、高质量的十二年一贯制公办学校，以标杆性'未来学校'为建设目标。学校秉承交大'饮水思源、爱国荣校'校训，深度依托交大多学科交融、理工见长的优势，凝练'科工通融、人文浸润'特色，以科技创新型人才为培养目标。中国教育学会副会长唐江澎任总督学，交大附中嘉定分校党总支书记曹建华任名誉校长。学校占地123亩，开办小学、初中和高中教育学段。
