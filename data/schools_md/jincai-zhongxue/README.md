# 上海市进才中学 (进才)

> 进才中学由台胞叶根林先生于1996年捐资兴建，是上海首批IB会员校。综评上线率89%，一本率78%以上。占地150亩，拥有游泳馆、天文台等一流设施，国际化办学特色鲜明。
