# 上海市扬子中学 (扬子中学)

> 上海市扬子中学于2004年经崇明区委区政府批准成立，校址为原崇明中学北门路旧址，是崇明区实验性示范性高中。学校占地8.4万平方米，建筑面积超6万平方米，设24个教学班、近千名在校生。以'扬帆教育'为办学特色，秉持'创造润泽生命的学校文化，让每一个师生扬起生命之帆'的办学理念，校训为'正气、锐气、灵气、大气'。'扬子文化千里寻'研学实践活动2018年获教育部经典工作经验奖。先后获上海市文明校园、全国篮球足球特色学校等荣誉。
