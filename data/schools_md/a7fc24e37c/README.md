# 上海浦东新区民办东鼎外国语学校 (浦东新区民办)

> 上海浦东新区民办东鼎外国语学校（前身为上海师范大学附属第二外国语学校）创建于2000年，2022年正式转设更名。学校位于迪士尼乐园、浦东国际机场与上海自贸区临港新片区交汇处，是一所十二年一贯制寄宿制民办双语学校。校舍占地近200亩，建筑面积约20万平方米。国内高中部近年高考本科上线率约90%以上，国际高中部QS世界排名前35大学录取率100%。开设日本EJU、加拿大OSSD、A-Level、新加坡、香港DSE等多方向课程。配备高尔夫球馆、网球馆、游泳池、冰球馆等场馆。
