# 上海市淞浦中学 (淞浦中学)

> 上海市淞浦中学创办于1985年，是淞宝西区唯一的一所公办高级中学，1998年迁入新校区。学校坐落于宝山区宝泉小区内，校园布局合理紧凑，假山水池、攀藤廊亭、石桌小径错落有致。教学设施先进，配有计算机校园网、有线电视系统、多功能演讲厅、室内体育馆等现代化设备。学校拥有师德高尚、业务精湛、年龄结构合理的教师队伍，培养了宝山区名师、学科带头人、青年教学尖子等多名骨干教师。是上海市科技特色学校、区文明单位、区行为规范示范学校，已列入全市五十所课程教改试点之一。
