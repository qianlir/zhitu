# 上海华旭双语学校 (华旭双语学校)

> 上海华旭双语学校（原上海华东师范大学附属双语学校）于2015年9月创办，是旭辉集团、上海国际汽车城（集团）有限公司与华师（上海）教育投资有限公司在嘉定安亭地区合作举办的K-12国际化民办寄宿制双语学校。已获IB世界学校正式授权，高中部提供IBDP、A-Level和IGCSE课程。是嘉定区首家K-12双语学校、首个IB项目学校。选修课程超过180门，师生比1:5，中外教比3:2，在校学生1900余人。师生来自12个区、全国17个省市及全球16个国家和地区。校训为'学为日进，人为日省'。
