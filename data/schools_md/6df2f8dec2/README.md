# 上海戏剧学院附属高级中学 (戏剧学院附属高级)

> 上海戏剧学院附属高级中学（简称上戏附中）创建于1925年，前身为培成女校，2004年由静安区政府与上海戏剧学院联合办学后更为现名。学校是静安区实验性示范性高中、上海市特色普通高中，以'戏剧艺术教育'为显著特色。开设戏剧影视表演、戏剧影视文学（含编导）、美术、播音与主持四个艺术专业课程，既为全国高等艺术院校输送高素质人才，也为综合类大学输送具有艺术特长的学生。知名校友包括学者余秋雨、作家程乃珊、演员肖雄等。
