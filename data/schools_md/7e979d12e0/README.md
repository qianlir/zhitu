# 上海市仙霞高级中学 (仙霞高级中学)

> 上海市仙霞高级中学成立于1988年，地处长宁区中西部，2009年被评为区实验性示范性学校。学校占地30余亩，校舍建筑面积近16000平方米。在多年办学中形成「团结奉献、艰苦奋进、严谨认真、开拓创新」的文化传统，在信息技术、双语教学、美术、足球等方面形成鲜明特色，培养具有现代信息素养和国际意识的学生。学校是全国足球特色学校、全国健康学校建设单位、上海市文明单位、上海市绿色学校。
