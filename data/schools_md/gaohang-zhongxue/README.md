# 上海市高桥中学 (高桥)

> 上海市高桥中学创办于1911年，坐落于浦东新区八百年古镇高桥，2024年3月正式被评为上海市实验性示范性高级中学。1946年成为上海市市立高桥中学，列18所市立中学之一；1960年被评为上海市重点中学。学校现有42个教学班、1727名在校学生（含6个新疆班214名新疆学生）。设计了'理趣——立交桥'课程结构，开发173门校本课程，重点建设自主体验式德育、环保创新科技、民族文化（家国情怀）三大特色课程群。校园内有明永乐帝亲笔'永乐御碑'。
