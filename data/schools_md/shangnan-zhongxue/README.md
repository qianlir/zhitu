# 上海市上南中学 (上南)

> 上海市上南中学创办于1963年，1987年被上海市教委命名为区重点中学，2008年首批通过浦东新区实验性示范性高中验收。学校现有30个教学班、1200名在校学生，拥有现代化天文台和天象馆。秉承'爱国、科学、人文'办学理念，形成了以天文科技为重点的科技教育特色、以'班级博客''教师魔灯'为重点的信息技术应用特色、以心理健康教育为重点的学校与社区互动特色。是全国艺术教育先进单位、市科技教育特色学校、全国心理健康教育先进学校。
