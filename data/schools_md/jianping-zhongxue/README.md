# 上海市建平中学 (建平)

> 建平中学始建于1944年，浦东新区龙头校。拥有特级教师10人居全市前三，师资质量排名全市第四。一本率稳定91%以上，裸考录取分常年全市最高之一。复交综评录取59人，进步速度排名全市第一。校风和谐友爱，文理均衡发展，1600名学生45个班。
