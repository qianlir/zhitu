# 上海市卢湾高级中学 (卢湾高级)

> 上海市卢湾高级中学创建于1953年，原为卢湾中学，是上海市人民政府首批创建的新型完全中学，1950年代被确立为上海市首批10所市重点中学之一。1999年原卢湾中学分拆为初高中，高中部即为今卢湾高中，2005年被命名为上海市实验性示范性高中。学校以'科学教育树人，人文精神立魂'为办学理念，科技教育特色彰显，荣获世界级奖项20项、国家级74项。拥有600人剧场、天文馆、天象馆等现代化设施，建筑面积3.5万平方米。
