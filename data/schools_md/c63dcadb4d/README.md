# 上海市中国中学 (中国中学)

> 上海市中国中学创办于1933年，由周慎修先生在日本侵华战争时期以'欲救亡图存，非教育不足以言此'之志兴办，以'中国'冠名。1936年建校三周年时，张学良、柳亚子等101位名流题词祝贺，校名由张学良题写，校训'智勇仁恕'由于右任书写。学校是徐汇区实验性示范性高中，秉承'爱国兴校'传统，以'责任教育'为核心理念。建有菁莪书院、天工坊等非遗传承专业教室，STEM和机器人实验室。获中国生态环保绿旗学校等荣誉。
