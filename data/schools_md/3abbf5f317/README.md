# 上海市江镇中学 (江镇中学)

> 上海市江镇中学地处东海之滨，毗邻浦东国际机场，创建于1948年，原名私立俊德中学。1956年改制为公立学校，1958年始设高中部，现为公办完全中学。学校占地83亩，建筑面积29000余平方米，拥有7幢教学楼和400米标准塑胶跑道运动场。现有36个教学班近1200名学生，教职工125人，其中上海市名师培养对象3人、种子计划教师2人。学校以'德育为先，学技俱长'为办学理念，持续推进楷模特色德育教育和足球办学特色。是全国首批校园足球布点学校、上海市体育项目传统学校，已向国家少年队和中超俱乐部输送十多名运动员。
