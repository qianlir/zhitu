# 上海市建平世纪中学 (建平世纪中学)

> 上海市建平世纪中学创办于2000年7月，是建平教育集团成员校。学校坐落于浦东新区行政文化中心，与上海科技馆、世纪公园毗邻，占地55亩，建筑面积14200平方米，是上海市花园单位。2009年初高中分离成为高级中学，2010年命名为浦东新区实验性示范性高中。学校具备招收外籍学生资质，是上海市心理健康教育示范校、上海市行为规范示范校。现有教工153名，其中特级教师1名、高级教师28名、区学科带头人和骨干教师共21名。拥有大型室内体育馆、人造草坪足球场、标准网球场等设施。
