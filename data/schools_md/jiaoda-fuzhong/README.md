# 上海交通大学附属中学 (交附)

> 交大附中理工科见长，与交大资源紧密联动，科创氛围浓郁。学校推行学生自主管理，在国际机器人大赛、头脑奥林匹克等赛事屡有斩获。创建于1954年，一本率常年85%以上，清北复交录取比例超30%。设有IB国际课程班。
