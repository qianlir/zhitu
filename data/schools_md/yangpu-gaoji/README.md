# 上海市杨浦高级中学 (杨高)

> 杨浦高级中学创建于1953年，1997年转制为杨浦高级中学，是上海市政府建设的11所现代化寄宿制高中之一。占地121亩，名誉校长于漪是全国首批语文特级教师。
