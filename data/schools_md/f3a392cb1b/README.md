# 上海市嘉定区封浜高级中学 (嘉定区封浜高)

> 上海市嘉定区封浜高级中学前身为1959年创建的嘉定县封浜中学，2012年撤一建二后高中部迁至现址。学校占地面积32462平方米，配备现代化教育设施，现有学生930余名，教职工100余人。在编教师83人，其中高级教师19名、一级教师31名，区级骨干教师15人。学校秉持'明理求知，健体尚美'校风，与高校合作打造'匠心素养'特色，获评上海市安全文明校园等荣誉。学校经历了从1959年封浜公社农业中学到1967年合并更名，1972年增设高中班的发展历程。
