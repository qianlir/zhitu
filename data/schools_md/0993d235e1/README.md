# 上海市民办平和学校 (民办平和学校)

> 上海市民办平和学校（上海市平和双语学校）创办于1996年9月，由上海金桥（集团）有限公司全额投资举办，坐落于浦东金桥碧云国际社区。校园占地37358平方米，建筑面积67698平方米。2001年起获准跨区县面向全市招生，2003年成为IBO组织成员学校，2004年被认定为上海市首批双语学校，2014年成为首批21所可开设高中国际课程的学校。学校秉承'平而不庸、和而不同'文化，培养具有国际视野的学习者。高中部开设IBDP课程，历届毕业生被哈佛、斯坦福、剑桥、牛津等世界名校录取。小学初中统考成绩名列浦东前茅。
