# 上海市松江区民办茸一中学 (松江区民办茸)

> 上海市松江区民办茸一中学创办于1993年，由松江区教育基金会主办，是上海市松江一中教育集团校。以'尚德、博学、求实、创新'为校训，秉持'立德树人，五育并举'教育理念。2004年搬迁至新校区，占地27亩，建筑面积13256平方米。学校开设小语种、媒介素养、英语戏剧、创新思维、人工智能等特色课程，搭建'乐思'课程体系。以信息化建设提升教学变革力，依托全场景教学和腾讯智慧校园推进课堂教学变革。是松江区拥有较高社会声誉的优质民办学校。注：该校原为完全中学，2007年后停止高中部招生，现为初中。
