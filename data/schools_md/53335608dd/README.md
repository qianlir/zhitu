# 上海市宝山区海滨中学 (宝山区海滨中)

> 上海市宝山区海滨中学前身为海滨初级中学，创建于1964年。1981年调整为吴淞区区属完全中学，1999年完成初高中脱钩成为高级中学。2001年成为宝山区第一所公立高中转制试点学校，更名为上海市和衷高级中学。2008年恢复公办性质，2010年通过区实验性示范性高中规划评审，2013年翻修完成后更名为上海市宝山区海滨中学。学校以'服务社会、服务家长、服务学生'为办学宗旨，恪守'和衷共济、开拓进取'校训，倡导'人人有才、人人成才'的教育理念，努力建设'小而精'的书香校园文化。现有在校师生800余名。
