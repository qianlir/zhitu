# 上海浦东新区民办康德学校 (浦东新区民办)

> 上海浦东新区民办康德学校（Shanghai Concord Bilingual School）位于浦东新区北蔡镇，2023年9月正式开学，是一所K-12一贯制民办双语寄宿制学校。总建筑面积67631平方米，占地33840平方米。高中部开设IGCSE/GCSE、A-level国际融合课程。康德品牌旗下学校已有四届高中毕业生，大部分被牛津、剑桥、帝国理工、伦敦政经、UCL等英国G5大学录取。学校践行'身心健康、德才兼备'教育思想，配备大型剧场、图书馆、室内游泳池、体育馆等设施。
