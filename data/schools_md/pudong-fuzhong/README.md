# 上海中学东校 (上中东)

> 上海中学东校位于临港新片区滴水湖畔，是上海中学与浦东新区合作开办的公办寄宿制完全中学。2023年高中部新校区启用，占地150亩，48个班规模。传承上中拔尖创新人才培养理念。
