# 上海市嘉定区嘉一实验高级中学 (嘉一实高)

> 上海市嘉定区嘉一实验高级中学于2022年9月正式开学，由嘉定一中创建，属于嘉一实验系学校。学校地处嘉定新城伊南社区，为公办区实验性示范性高中。学校与嘉定一中实行一体化管理，教职工考核、岗位聘任统一，共享嘉定一中的优质教育资源和师资力量。嘉定一中创办于1926年，1958年被命名为首批上海市重点中学，2005年被命名为上海市实验性示范性高中，以'人文科技合力，培育嘉懿学子核心素养'为特色，以'合作—创新'教育为办学理念。校训为'追踪前哲、认识时代'。
