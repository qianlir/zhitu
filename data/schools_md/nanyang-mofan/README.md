# 上海市南洋模范中学 (南模)

> 南洋模范中学创建于1901年（南洋公学附属小学），是中国人自己创办的最早新式学堂之一。文艺体育排名全市第一，青蜂班口碑和升学实力出众。复交综评录取率约12%，近年与七宝、建平同属八校前三。徐汇区头牌高中，百年底蕴深厚。
