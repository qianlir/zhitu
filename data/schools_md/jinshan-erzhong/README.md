# 上海市金山区第二中学 (金山二中)

> 未找到名为'上海市金山区第二中学'的独立学校。金山区带'第二'或'二'字的高中主要为上海师范大学第二附属中学（上师大二附中），前身是1976年创建的石化一中，1985年由金山区政府、上海石化和上海师大三方联办更名。2009年成为金山区实验性示范性高中，2022年获评上海市特色普通高中。学校以'生态科技教育'为特色，建有生态学创新实验室，获评国际生态学校、联合国教科文组织中国可持续发展项目实验学校。占地70亩，现有在校生约1200名。如此校名有误，请核实后补充。
