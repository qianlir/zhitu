# 上海民办民一中学 (民办民一中学)

> 上海民办民一中学创办于2001年，注册于2007年，是崇明区一所涵盖初中和高中的民办完全中学，崇明中学紧密型教育集团成员校。学校总占地面积33800平方米，建筑面积23300平方米，两个校区分别位于堡镇解放街230号和城桥镇东门路557号。现有28个教学班（初中16班、高中12班），专任教师90多名，中高职称占80%以上。中考成绩稳居崇明全区之首，市重点录取率稳定在65%以上。先后获上海市民办学校规范管理5A级单位、上海市民办特色学校、上海市学生行为规范示范校等荣誉。
